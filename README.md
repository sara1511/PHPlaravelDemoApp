# PHPlaravelDemoApp
A sample code of PHP with Laravel

## Deploy this app to localhost

```
$ git clone https://github.com/sara1511/PHPlaravelDemoApp.git
$ cd PHPlaravelDemoApp/
$ composer install
$ php artisan serve
```

Access to

- http://127.0.0.1:8000/
- http://127.0.0.1:8000/hello/
- http://127.0.0.1:8000/world/
